Sales Performance Dashboard | Power BI

Project Overview

This project was completed as part of my Data Analytics Internship at SyntaxHub.

The objective of the project was to transform raw sales transaction data into an interactive business intelligence dashboard capable of tracking performance, identifying trends, and supporting data-driven decision making.

The project involved data preparation, KPI development, dashboard design, data visualization, and business insight generation using Microsoft Power BI.

---

Business Problem

Organizations generate large volumes of transactional sales data, but raw data alone does not provide meaningful insights.

The goal of this project was to create a centralized dashboard that enables stakeholders to:

- Monitor overall business performance
- Track revenue and profitability
- Analyze product performance
- Understand customer activity
- Identify sales trends over time
- Interactively filter and explore data

---

Dataset Information

The dataset contains sales transaction information including:

- Order ID
- User ID
- Product ID
- Revenue Data
- Cost of Goods Sold (COGS)
- Profit Data
- Purchase Quantity
- Transaction Date

The data spans multiple years (2012–2015), making it suitable for trend analysis and performance monitoring.

Data Preparation

Several preprocessing and validation steps were performed before dashboard development:

Data Validation

- Reviewed dataset structure
- Verified column data types
- Checked for inconsistencies

Data Cleaning

- Validated numerical fields
- Verified date fields
- Reviewed missing values
- Ensured data quality for reporting

Data Modeling

- Imported dataset into Power BI
- Configured relationships where necessary
- Optimized fields for reporting and analysis

---

DAX Measures Created

The following key measures were developed:

Total Revenue

Calculates overall sales revenue.

Total Orders

Counts total orders processed.

Total Customers

Calculates unique customers using DISTINCTCOUNT.

Total Profit

Measures total business profit.

Profit Margin %

Calculates profitability relative to revenue.

---

Dashboard Components

Executive KPI Section

The dashboard provides a high-level business summary through:

- Total Revenue
- Total Orders
- Total Customers
- Total Profit
- Profit Margin %

These KPIs enable rapid performance assessment.

---

Product Performance Analysis

Revenue by Product

Identifies products generating the highest revenue.

Profit by Product

Highlights products contributing most to profitability.

This comparison allows identification of:

- High-revenue products
- High-profit products
- Potential underperforming products

---

Revenue Trend Analysis

A time-series visualization was developed to analyze revenue performance between 2012 and 2015.

This enables stakeholders to:

- Monitor growth patterns
- Detect declines
- Evaluate business performance over time

---

Interactive Filtering

The dashboard includes slicers allowing users to dynamically explore:

- Product Performance
- Yearly Performance

This improves usability and supports deeper analysis.
